» 使用说明
一般使用
  1. 打开 wp-content/themes/<您的模板目录>/index.php

    也可以是 archive.php, single.php, post.php 或者 page.php。

  2. 找到:
    <?php while (have_posts()) : the_post(); ?>
  3. 在它下面任何地方加上：
    <?php if(function_exists('the_views')) { the_views(); } ?>
  4. 选择'WP-Admin -> Settings -> PostViews' 对插件进行设置。
浏览统计 (使用Widgets)
  1. 选择'WP-Admin -> Appearance -> Widgets'
  2. 找到浏览。
浏览统计 (WP循环以外)
  • 要显示浏览次数最少的文章
  • 用:
    <?php if (function_exists('get_least_viewed')): ?>
       <ul>
          <?php get_least_viewed(); ?>
       </ul>
    <?php endif; ?>

    第一个参数是 'post', 'page' 或者 'both'。
    第二个参数表示文章数量。

    默认: get_least_viewed('both', 10);

  • 要显示浏览次数最多的文章
  • 用:
    <?php if (function_exists('get_most_viewed')): ?>
       <ul>
          <?php get_most_viewed(); ?>
       </ul>
    <?php endif; ?>

    第一个参数是 'post', 'page' 或者 'both'。
    第二个参数表示文章数量。

    默认: get_most_viewed('both', 10);

  • 要显示按标签浏览次数最少的文章
  • 用:
    <?php if (function_exists('get_least_viewed_tag')): ?>
       <ul>
          <?php get_least_viewed_tag(); ?>
       </ul>
    <?php endif; ?>

    第一个参数是标签id。
    第二个参数是 'post', 'page' 或者 'both'。
    第三个参数表示文章数量。

    默认: get_least_viewed_tag(1, 'both', 10);

  • 要显示按标签浏览次数最多的文章
  • 用:
    <?php if (function_exists('get_most_viewed_tag')): ?>
       <ul>
          <?php get_most_viewed_tag(); ?>
       </ul>
    <?php endif; ?>

    第一个参数是标签id。
    第二个参数是 'post', 'page' 或者 'both'。
    第三个参数表示文章数量。

    默认: get_most_viewed_tag(1, 'both', 10);

  • 要显示按分类浏览次数最少的文章
  • 用:
    <?php if (function_exists('get_least_viewed_category')): ?>
       <ul>
          <?php get_least_viewed_category(); ?>
       </ul>
    <?php endif; ?>

    第一个参数是分类id。
    第二个参数是 'post', 'page' 或者 'both'。
    第三个参数表示文章数量。

    默认: get_least_viewed_category(1, 'both', 10);

  • 要显示按分类浏览次数最多的文章
  • 用:
    <?php if (function_exists('get_most_viewed_category')): ?>
       <ul>
          <?php get_most_viewed_category(); ?>
       </ul>
    <?php endif; ?>

    第一个参数是分类id。
    第二个参数是 'post', 'page' 或者 'both'。
    第三个参数表示文章数量。

    默认: get_most_viewed_category(1, 'both', 10);

  • 要对浏览次数最多/最少文章排序
  • 用:
    <?php query_posts('v_sortby=views&v_orderby=desc') ?>

    或者将变量放到url中:

    http://yoursite.com/?v_sortby=views&v_orderby=desc

    如果是浏览次数最少文章，则将 desc 替换成 asc。
