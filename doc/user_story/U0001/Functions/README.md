### 捐助 
- 快速搜索
  当用户点击捐助菜单的时候, 页面右侧的搜索栏滑出, 用户在搜索栏顶部的快速搜索框输入捐助编号
  系统在用户输入时开始匹配输入的编号，如果有找到多个模糊匹配的记录，就在搜索框下部列出， 如果只有
  一个匹配的记录，就直接跳转到相应的捐助查看页面
  
- 高级搜索
  
- 查看所有捐助
